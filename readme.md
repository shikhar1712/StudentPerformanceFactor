# Student Performance Factors Analysis using Python

## Overview
This project explores the factors that influence student exam performance using Python for data analysis and visualization. The analysis focuses on identifying relationships between exam scores and variables such as study hours, attendance, sleep, parental involvement, motivation, previous scores, and access to resources.

## Objective
The main objective of this project is to understand which academic, personal, and environmental factors are associated with better student performance.

## Dataset
- Source: Kaggle
- Dataset: Student Performance Factors
- Link: https://www.kaggle.com/datasets/lainguyn123/student-performance-factors

## Tools and Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Project Workflow
1. Loaded and explored the dataset
2. Cleaned column names and checked data quality
3. Performed exploratory data analysis (EDA)
5. Visualized relationships between exam scores and performance factors
6. Summarized key findings

## Key Analyses Performed
- Distribution of exam scores
- Correlation analysis of numerical variables
- Study hours vs exam score
- Attendance vs exam score
- Sleep hours vs exam score
- Motivation level vs exam score
- Parental involvement vs exam score
- Internet access vs exam score
- Teacher quality vs exam score
- Previous scores vs exam score

## Key Insights
- Exam scores appear to be positively related to hours studied and attendance.
- Previous scores show a strong relationship with current exam scores.
- Students with higher motivation levels tend to achieve better marks.
- Students with good amount of sleep tend to have better marks.
- Better parental involvement and access to resources may support improved performance.
- Internet access and teacher quality also appear to influence student outcomes.
- Motivation level, parental involvement, and access to resources show noticeable differences in exam scores.

## Conclusion
This project demonstrates how data analysis can be used to understand the factors affecting academic performance. It helped strengthen skills in data cleaning, exploratory data analysis, visualization, and insight generation using Python.
